Legend Of Warcraft

Java 8 - LibGDX

Legend of Warcraft is an immersive 2D RPG developed in Java using the LibGDX framework, seamlessly blending open-world exploration with strategic combat. Players navigate a vast map designed with Tiled, engaging in a mix of real-time skirmishes against monsters and tactical turn-based battles against bosses. The game features a robust progression system complete with leveling, experience tracking, and inventory management, challenging players to collect keys and complete quests to unlock these decisive boss encounters.
